Rube Goldberg Machine

A Rube Goldberg machine inspired by a childhood dream, where a marble travels through four distinct fantasy worlds: each reflecting a different season, color palette, and mood.

A walkthrough of the Rube Goldberg interactive worlds and seasonal transitions.

Description

Rube Goldberg Machine is a 3D animated interactive project created using Unity software, where a marble travels through four dreamlike worlds, each representing a different season. The work merges storytelling, physics, and motion design to visualize how small actions can create playful, chain-reaction narratives.

Concept

This project was inspired by a childhood dream where each room transformed into a new environment. I reimagined that concept by creating four interconnected worlds: Candyland (Spring), Fall World, Summer World, and Winter World. Each stage reflects a different mood and uses color, sound, and rhythm to guide the viewer through a continuous flow of motion.

Design Process

I first white-boxed the environment to test the mechanics, then refined each world’s lighting and textures to reflect its emotional tone. Pastel pinks for Candyland evoke childhood nostalgia, orange hues for Fall feel warm and cinematic, bright blues for Summer create energy, and icy whites for Winter feel serene and reflective.

Reflection

This project originally started as a story about my daily commute, but after losing my progress twice, I decided to reinvent it. That setback ended up being a creative turning point, because it pushed me toward a more imaginative and personal direction. I learned to embrace accidents, reframe frustration as inspiration, and build worlds that reflect emotion through motion.
